50音ボタンと自動入力

最速フロー

ヨミ（50音）を選んだ際、該当者が1人なら即座に名前をセット。無駄なタップを削減します。

Initial Value (利用者名):

IF(
  COUNT(FILTER("利用者リスト", [ヨミ行頭] = [_THISROW].[ヨミ])) = 1,
  ANY(FILTER("利用者リスト", [ヨミ行頭] = [_THISROW].[ヨミ])),
  ""
)

データ管理の要 (KEY)

重要設定

データの不整合を防ぐため、_RowNumber ではなく、独自に作成した ID を KEY として設定。表示には 利用者名 を LABEL として使用します。

出欠の自動振り分け (Slices)

整理整頓

スライス機能を利用して、記録を保存した瞬間に「出席者」と「欠席者」を別々のタブへ自動で仕分けます。

  • 出席者条件: [出欠] = "出席"
  • 欠席者条件: [出欠] = "欠席"

※欠席者ビューは「記録日時の降順（最新順）」で並び替える設定を適用済みです。

視覚的な警告 (Format Rules)

安全管理

一目で異常を察知できるよう、条件付き書式を設定。

  • 高熱アラート: NUMBER([体温]) >= 37.5 で赤字表示。
  • 出欠強調: 欠席は赤、出席は緑で色分け。
  • 記録日時: 画面右上のサマリー列に配置し、最新記録を即座に把握。
